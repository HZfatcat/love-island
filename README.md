# 🏝️ Love Island — 我们的甜蜜小岛

> 基于 Animal Crossing 风格的情侣专属甜蜜小站 💕

## ✨ 功能

- 🏠 **首页** — 恋爱天数 + 纪念日倒计时 + 今日心情记录
- 📖 **恋爱日记** — 记录每一天的甜蜜瞬间
- 🌟 **心愿清单** — 一起许愿，一起实现
- 📸 **回忆相册** — 我们的照片记忆墙
- 📅 **纪念日** — 重要日子的倒计时
- 💌 **留言板** — 模拟聊天互动

## 🎨 设计风格

灵感来源于 [animal-island-ui](https://atomgit.com/guokaigdg/animal-island-ui) 的动物森友会风格 UI：
- 暖色调羊皮纸背景
- 薄荷绿主色 + 圆润胶囊按钮
- 3D 像素阴影（按下下沉效果）
- 波点纹理卡片
- 漂浮爱心装饰

## 🚀 部署到 GitHub Pages

这个仓库已经配置好，只需几步即可发布：

### 方式一：网页操作（推荐）

1. 打开 [github.com/new](https://github.com/new) 创建新仓库
2. 仓库名填 `love-island`，选 **Public**
3. 创建后按页面提示的「…or push an existing repository」下的命令操作

### 方式二：命令行

```bash
# 在本地执行
git remote add origin https://github.com/<你的用户名>/love-island.git
git push -u origin main
```

### 开启 Pages

1. 进入仓库 → **Settings** → **Pages**
2. Source 选 **Deploy from a branch**
3. Branch 选 `main`，文件夹选 `/ (root)`
4. 点击 **Save**
5. 等待几分钟，访问 `https://<你的用户名>.github.io/love-island/` 🎉

## 🔧 本地使用

直接双击 `index.html` 即可在浏览器中打开预览，无需安装任何东西。
